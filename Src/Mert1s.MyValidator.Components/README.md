# Mert1s.MyValidator.Components

ASP.NET Core components and integrations for `Mert1s.MyValidator`.

This project provides UI helpers and Blazor integration components that make it easier to integrate validation into Blazor forms and components.

Usage
--
Register the DI package (from the DependencyInjection package) and use the components in your Blazor pages.

Packaging
--
This project includes a `README.md` and will include it in the published NuGet package. If an `icon.png` file exists in the project folder, it will be included as the package icon.
